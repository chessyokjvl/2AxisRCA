JVL RCA System
คุณสมบัติ 
- Dashboard แสดงข้อมูลสำคัญของอุบัติการณ์ระดับ E ขึ้นไปทั้งหมด ทั้งเชิงปริมาณและเชิงคุณภาพ ในการทำ RCA และการแก้ไขด่านกั้นที่ล้มเหลว
- แสดงทะเบียนอุบัติการณ์ทั้งหมด พร้อมสถานะการทำ RCA และด่านกั้นที่ล้มเหลว (หากทำ RCA แล้ว) พร้อมบันทึก update และแสดงสถานะการแก้ไข ด่านกั้นตามต้องการ
- เครื่องมือทำ RCA แบบ 2-Axis พร้อม 4 Tier 10 กลุ่ม 35 ปัจจัย 85 ด่านกั้น สามารถบันทึก แก้ไข แบบ Interactive พร้อมทั้งบันทึกเก็บเข้าฐานข้อมูล และ Generate เป็น Diagram 2-Axis with 4-Tier ได้อัตโนมัติ
- มีระดับการเข้าถึง 3 Level Admin ทำได้ทุกอย่าง User สำหรับบุคลากรโรงพยาบาลที่ใช้ในการทำ RCA และบันทึกข้อมูลต่างๆ และ Guest สำหรับบุคคลภายนอกที่อยากจะลองเล่นดูเท่านั้น (ลองใช้แต่ไม่บันทึก)
